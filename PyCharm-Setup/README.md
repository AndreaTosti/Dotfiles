    * Color Scheme

          * Scheme: [Darcula Darker](https://plugins.jetbrains.com/plugin/index?xmlId=com.vecheslav.darculaDarkerTheme)
          * [ ] Use color scheme font instead of the default

          * [ ] Use color scheme font instead of the default
